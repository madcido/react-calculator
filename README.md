## Installation Guide
### Prerequisities
System dependencies:
```
$ node -v
  v10.15.3

$ npm -v
  6.4.1

$ webpack -v
  4.34.0
```

#### Local setup
##### Clone project
```
$ git clone https://github.com/madcido/react-calculator.git
$ cd ./react-calculator
```
##### Install dependencies
```
$ npm install
```
##### Build project
```
$ npm start
```
